## Getting Started

1. Install the pods

```
pod update
```

2. Open _FirebaseStarterSwiftUIApp.xcworkspace_ in Xcode
3. Build and run

## Features

- Login with Email and Password
- Login & Registration
- Firestore Integration
- Logout
